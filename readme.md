# Pothos Filter

## Installation

```bash
git clone git@github.com:noxify/pothos-filter.git
cd pothos-filter
npm i
```

## Start the server

To run the server in development mode:

```bash
npm run dev
```

Otherwise, use:

```bash
npm run start
```
